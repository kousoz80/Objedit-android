## Objedit-android
  
本プログラムはのandroid端末上で動作する[ObjectEditor](https://github.com/kousoz80/ObjectEditor) です。
  
##### [起動画面]
  
![起動画面](https://i.imgur.com/gn7yZ3l.png)
  
androidアプリの他にC/C++言語、BASIC言語等のプログラム開発もできます。
  
またGUIデザイナーも備えていますので簡単にGUIプログラムを作成することができます。
